# Visual Generation Contest - Team 06

## How to Run
1. Install dependencies:
   `pip install -r code/requirements.txt`

2. Run the generation script:
   `python code/main.py`

3. Check `outputs/` for the results.

## Concept
We use Factorized Diffusion to combine "Prompt A" (Low Freq) and "Prompt B" (High Freq).
